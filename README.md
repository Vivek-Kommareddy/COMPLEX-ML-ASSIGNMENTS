# COMPLEX-ML-ASSIGNMENTS

## 📌 Overview
This repository contains a collection of **Machine Learning and Statistical Modeling assignments** implemented using **Python and Jupyter Notebooks**.  
It demonstrates core concepts ranging from probability theory to advanced models like **Support Vector Machines (SVM)** and **Neural Networks (CNNs)**.

---

## 🚀 Key Highlights
- Implementation of fundamental ML algorithms from scratch and using libraries  
- Hands-on exploration of **probability and statistical methods**  
- Practical understanding of **supervised and unsupervised learning techniques**  
- Covers both **theoretical concepts and applied machine learning**

---

## 🛠️ Tech Stack
- Python  
- Jupyter Notebook  
- NumPy  
- Pandas  
- Matplotlib / Seaborn  
- Scikit-learn  
- (Optional) TensorFlow / PyTorch for deep learning

## ⚙️ Installation & Setup

### 1. Clone the repository
```
git clone https://github.com/Vivek-Kommareddy/COMPLEX-ML-ASSIGNMENTS.git
cd COMPLEX-ML-ASSIGNMENTS
```

### 2. Install dependencies
```
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

### 3. Run Jupyter Notebook
```
jupyter notebook
```
